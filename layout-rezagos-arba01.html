Layout Rezagos — ARBA01
Herramienta de gestión de posiciones · Logística Inversa · Mercado Libre
¿Para qué sirve esta herramienta?
Permite registrar y visualizar en tiempo real qué pallet está en cada posición del layout de rezagos no sistémicos de ARBA01.
Reemplaza el Google Form y actualiza automáticamente la base de datos de Google Sheets.
📥
Cómo ALMACENAR un pallet
1
Tocá el botón ALMACENAR en la pestaña Operar
2
Escaneá el código del pallet (el campo se llena solo y avanza automáticamente)
3
Escaneá la posición de destino (ej: DOC-5-001)
4
Seleccioná la dimensión (Pallet / Bulto / Voluminoso) y tocá CONFIRMAR
💡 El scanner hace Enter automático — no necesitás tocar la pantalla entre escaneos.
📤
Cómo VACIAR una posición
1
Tocá el botón VACIAR en la pestaña Operar
2
Escaneá la posición a vaciar — se muestra el pallet actual en esa posición
3
Confirmá tocando VACIAR POSICIÓN
🗺️ Layout
Vista visual de todas las posiciones de ARBA01.

● Verde = libre
● Rojo = ocupado

Tocá cualquier posición para ver el detalle o realizar una acción.
🔍 Buscar
Localizá cualquier pallet o posición.

Podés buscar por:
· Código de pallet (ej: RZ-1096615)
· ID de posición (ej: DOC-5)

El resultado muestra ubicación, estado y quién lo registró.
